🎯 How to Use
Option 1: Try Sample Code
  1. Go to http://localhost:5000
  2. Click "Poor", "Moderate", or "Excellent"
  3. Click "Analyze & Get Green Suggestions"
  4. See results with all metrics
Option 2: Write Your Own
  1. Click "Clear" button
  2. Paste your Python code
  3. Click "Analyze"
  4. View detailed analysis
