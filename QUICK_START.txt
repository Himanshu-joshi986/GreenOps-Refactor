## ❓ COMMON QUESTIONS

**Q: What means a good Green Score?**
A: 80+ is excellent, 60-80 is good, 40-60 needs improvement

**Q: Why is my score low?**
A: Usually due to lots of loops without vectorization. Use NumPy/Pandas instead.

**Q: Can I analyze large code?**

## 🎯 OPTIMIZATION TIPS

### Quick Wins
1. Use list comprehensions instead of for loops
2. Use NumPy/Pandas for data operations
3. Use async/await for I/O-heavy code
4. Avoid deep recursion

### For Different Scenarios

**For Web Services:**
- Optimize database queries
- Use async/await for API calls
- Cache results when possible

**For Data Processing:**
- Use Pandas vectorization
- Avoid iterrows()
- Use NumPy for numerical operations

**For ML/AI:**
- Use batch processing
- Consider model quantization
- Use inference optimization
